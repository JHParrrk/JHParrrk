# ⚾ 마운드 위 투수의 마음으로 코드를 쓰는 박준하입니다.

> **"1%의 실투를 줄이기 위해 단련하는 투수처럼, 치밀한 설계와 최적화로 소프트웨어의 변수를 통제합니다."**

단순히 동작하는 코드를 넘어, **데이터 정합성**과 **시스템 가용성**을 최우선으로 생각하는 백엔드 중심 풀스택 엔지니어입니다. AI가 생성하는 코드를 넘어선 **인간 엔지니어의 '최적화' 가치**를 증명하기 위해 끊임없이 학습하고 기록합니다.
